# Contributing to homebrew-bottube

Homebrew tap for BoTTube. Contributions welcome!

## How to Contribute

### Adding or Updating Formulas
1. Fork this repository
2. Create a branch for your change
3. Test the formula locally: `brew install --formula bottube.rb`
4. Run `brew audit` and `brew style` on your formula
5. Open a Pull Request

### Formula Guidelines
- Follow [Homebrew formula conventions](https://docs.brew.sh/Formula-Cookbook)
- Include proper checksums for all downloads
- Test on both Intel and Apple Silicon where possible
- Keep descriptions concise

### Reporting Issues
- Include your macOS version and Homebrew version
- Show the full error output
- Try `brew doctor` first

## Questions?

Open an issue on this repo.
